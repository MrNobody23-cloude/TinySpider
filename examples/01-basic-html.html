📊 Insight Analytics Example

This page demonstrates basic Insight Analytics integration with plain HTML.

ℹ️ Configuration:

Site ID: demo-site

Endpoint: http://localhost:3000/collect

The tracker is loaded via script tag and automatically tracks page views and clicks.

📈 What's Being Tracked:
  • ✅ Page views (automatic)
  • ✅ Link clicks (automatic)
  • ✅ Button clicks (automatic)
  • ✅ Custom events (when you click the button)
  • ✅ Referrer information
  • ✅ User session
